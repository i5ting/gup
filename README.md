gup
===

a npm wrapper for git utils

[![npm version](https://badge.fury.io/js/gup.svg)](http://badge.fury.io/js/gup)

## Install 

	npm install -g gup
	
## Usage

### 无脑 pull 当前分支
执行`gup`命令即可

	gup
	
### 在 git 中找回丢失的 commit

	fsck
